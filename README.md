# GENESYS Agency — Tunnel de Diagnostic E-Commerce

Tunnel de qualification interactif qui identifie en 3 minutes le profil d'un entrepreneur e-commerce, son blocage actuel, et le route vers la solution adaptée (communauté GENESYS Skool ou appel diagnostic personnalisé).

## Stack

- HTML / CSS / JavaScript vanilla
- Aucune dépendance externe
- Persistence localStorage
- Diagnostic généré localement par templates conditionnels (en production : API Anthropic)

## Local

Ouvrir `index.html` dans un navigateur moderne (Chrome, Safari, Firefox).

## Deployment

GitHub Pages — branche `main`.
